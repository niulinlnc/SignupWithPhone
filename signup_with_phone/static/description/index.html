Python Library Installation

To install this module; make sure, you have following things installed on Ubuntu.
- sudo pip install phonenumbers

Allow Users For Signup

Settings -> General Settings -> Click on Allow external users to sign up -> Apply

Extend Signup/Register

Added Extra field(Mobile) in Signup/Register Form.
User can Signup with either Email or Mobile.
On Signing up it will create new User and Partner with respective Email or Mobile fields.

Validation in mobile

User has to add mobile number with country code and a reference example is provided in the field itself.

Sign in Screen

User can also signup with mobile number instead of an email.
