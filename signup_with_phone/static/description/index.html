Python Library Installation

To install this module, you need to have installed.
On Ubuntu::
- sudo pip install phonenumbers

Allow Users For Signup

Settings -> General Settings -> Click on Allow external users to sign up -> Apply

Extend Signup/Register

Added Extra field(Mobile) in Signup/Register Form.
User can Signup with either Email or Mobile.
On Signup it will be create User and Partner with Email/Mobile fields.

Validation in mobile

Add Mobile number with country code for Referance purpose add placeholder in Signup Form.

Sign in Screen

If user has Sign up with Mobile number, then can login with Mobile too.
